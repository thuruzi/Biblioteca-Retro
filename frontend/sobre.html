Sobre o Projeto

O Retro Library é um sistema web desenvolvido como trabalho acadêmico de faculdade para a disciplina de Programação Web. O objetivo do projeto é demonstrar, de forma prática, a criação de um CRUD completo utilizando HTML5, CSS3, JavaScript Vanilla, PHP puro orientado a objetos, PDO e banco de dados MySQL/MariaDB.

A proposta do sistema é simular uma biblioteca de jogos retrô, permitindo cadastrar, listar, consultar, editar e excluir informações de jogos clássicos. O projeto também possui um modo de testes com localStorage, para que as interfaces possam ser avaliadas mesmo quando o banco de dados não estiver configurado.
